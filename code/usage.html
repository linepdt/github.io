使用说明

• 会员是否支持跨平台？

会员支持跨平台、跨设备多端数据备份同步

• 条形码

一般用在实体商品上商品码，通常是13位，具有唯一性，我们生成条形码不要超过15位

• 二维码

可容纳较多字符的码制，记录的信息越多，二维码越复杂且不易识别，通常我们不建议生成超过300个中文或600个英文字符的二维码

• 名片

我们采用国际标准的格式生成名片二维码，可被系统相机或者微信等社交软件扫描自动识别姓名、手机号码、电子邮箱、联系地址等信息

• WiFi

生成的二维码可通过系统相机扫描识别自动连接对应的WiFi无线网络

• 短信

生成的二维码可通过系统相机扫描会建议进入短信息界面发短信

• 手机号码

生成的二维码可通过系统相机扫描会建议进入拨号界面

• 应用地址

生成的二维码可通过系统相机扫描后会建议进入App Store对应的App下载页面

• 批量生成二维码

用于企业为其工业产品批量有序地生成各自的二维码，支持前缀、后缀等参数的插入，文件名自动对应内容，生成1000个二维码不到一分钟即可完成，可通过分享的方式批量压缩后发送到桌面

• 关于会员

永久会员享有应用程序内支持的会员特权，无时间限制，仅须一次购买。其他时长会员在订阅期内享有相应时长的会员特权，重复购买(不建议)会自动叠加会员时长，如果你卸载后重新安装了软件，可以通过恢复购买来恢复会员特权。对于一次性奖励品、消耗品不支持恢复！对奖励品作弊会取消相应权利!
